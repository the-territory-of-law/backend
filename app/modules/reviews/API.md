# Reviews API

Документация по отзывам после сделок.

Статус: **предлагаемый API (draft, HTTP ещё не реализован)**.

Базовый префикс: `/reviews`.

Отзыв привязан к завершённой сделке. Автор — клиент (`client_id`), объект оценки — юрист (`lawyer_id`).

---

## Оставить отзыв

```http
POST /reviews
```

После сделки в статусе `completed`.

### Тело запроса

```json
{
  "deal_id": 201,
  "rating": 5,
  "text": "Профессиональная работа, всё в срок"
}
```

| Поле | Тип | Обязательное | Описание |
|---|---|---|---|
| `deal_id` | `integer` | Да | ID завершённой сделки |
| `rating` | `integer` | Да | Оценка `1`–`5` |
| `text` | `string` | Нет | Текст отзыва |

### Ответ `201`

```json
{
  "id": 12,
  "deal_id": 201,
  "client_id": 20,
  "lawyer_id": 10,
  "rating": 5,
  "text": "Профессиональная работа, всё в срок",
  "created_at": "2026-05-15T10:00:00Z"
}
```

### Ошибки

| Код | Описание |
|---|---|
| `400` | Сделка не завершена или отзыв уже есть |
| `401` | Не авторизован |
| `403` | Только клиент сделки |
| `404` | Сделка не найдена |
| `422` | `rating` вне диапазона 1–5 |

## Получить отзыв

```http
GET /reviews/{review_id}
```

### Ответ `200`

```json
{
  "id": 12,
  "deal_id": 201,
  "client_id": 20,
  "lawyer_id": 10,
  "rating": 5,
  "text": "Профессиональная работа, всё в срок",
  "created_at": "2026-05-15T10:00:00Z"
}
```

### Ошибки

| Код | Описание |
|---|---|
| `404` | Отзыв не найден |

---

## Список отзывов

```http
GET /reviews
```

### Query-параметры

| Параметр | Тип | Описание |
|---|---|---|
| `lawyer_id` | `integer` | Отзывы о юристе (user id) |
| `client_id` | `integer` | Отзывы клиента |
| `deal_id` | `integer` | Отзыв по сделке |
| `rating_gte` | `integer` | Минимальная оценка |
| `limit` | `integer` | Лимит |
| `offset` | `integer` | Смещение |

### Ответ `200`

```json
{
  "reviews": [
    {
      "id": 12,
      "deal_id": 201,
      "client_id": 20,
      "lawyer_id": 10,
      "rating": 5,
      "text": "Профессиональная работа",
      "created_at": "2026-05-15T10:00:00Z"
    }
  ],
  "total": 24,
  "has_more": true
}
```

### Ошибки

| Код | Описание |
|---|---|
| `400` | Некорректные фильтры |

---

## Обновить отзыв

```http
PATCH /reviews/{review_id}
```

Только автор, в течение ограниченного окна (например, 48 часов).

### Тело запроса

```json
{
  "rating": 4,
  "text": "Хорошо, но были задержки по срокам"
}
```

### Ответ `200`

Объект отзыва (как в `GET`).

### Ошибки

| Код | Описание |
|---|---|
| `400` | Срок редактирования истёк |
| `401` | Не авторизован |
| `403` | Не автор |
| `404` | Не найден |
| `422` | Валидация |

---

## Удалить отзыв

```http
DELETE /reviews/{review_id}
```

Автор или admin.

### Ответ `204`

Пустое тело.

### Ошибки

| Код | Описание |
|---|---|
| `401` | Не авторизован |
| `403` | Нет прав |
| `404` | Не найден |

---

# Структуры данных

```ts
type Review = {
  id: number
  deal_id: number
  client_id: number
  lawyer_id: number
  rating: number
  text: string | null
  created_at: string
}

type ReviewsPage = {
  reviews: Review[]
  total: number
  has_more: boolean
}
```
